# e-plantShopping

e-plantShopping is an online plant shopping application called Paradise Nursery.

## Features

- Browse plants by category
- View plant details with image, name, and price
- Add plants to cart
- Increase or decrease quantity
- Remove items from cart
- View total cart amount
- Redux state management

## Technologies Used

- React
- Vite
- React Router
- Redux Toolkit

## Run Locally

```bash
npm install
npm run dev
